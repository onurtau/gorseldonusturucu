### Çözüm - 3 Buton Yapısı:

1. **Giriş Yap (Login)** - Outline stil, mor çerçeve
2. **Kayıt Ol (Register)** - Outline stil, mavi çerçeve  
3. **👑 Premium** - Altın gradient + Crown ikonu

### Yapılan Değişiklikler:

#### 1. Crown İkonu Import Edildi ✅
```jsx
import { Globe, ArrowLeft, Download, Menu, X, Crown } from 'lucide-react';
//                                                ^^^^^ YENİ
```

#### 2. Register Butonu Eklendi ✅
- **Türkçe:** "Kayıt Ol"
- **İngilizce:** "Sign Up"
- **Stil:** Mavi outline (`border-2 border-blue-600 text-blue-600`)
- **Hover:** `hover:bg-blue-50`
- **Font:** `font-semibold`

**Kod:**
```jsx
<button
  onClick={() => {
    onRegisterClick();
    setIsMobileMenuOpen(false);
  }}
  className="w-full px-4 py-2.5 border-2 border-blue-600 text-blue-600 hover:bg-blue-50 rounded-lg font-semibold transition-all text-sm"
>
  {language === 'tr' ? 'Kayıt Ol' : 'Sign Up'}
</button>
```

#### 3. Premium Butonu Güncellendi ✅
- **Yazı:** "Premium" (kısa ve net, "Premium'a Geç" kaldırıldı)
- **İkon:** Crown (👑) - taç simgesi
- **Renk:** Altın gradient (`from-yellow-500 to-orange-500`)
- **Gölge:** `shadow-lg shadow-yellow-500/30`
- **İkon Rengi:** Beyaz (`text-white`)

**Kod:**
```jsx
<button
  onClick={() => {
    onRegisterClick();
    setIsMobileMenuOpen(false);
  }}
  className="w-full flex items-center justify-center gap-2 px-4 py-2.5 bg-gradient-to-r from-yellow-500 to-orange-500 hover:from-yellow-600 hover:to-orange-600 text-white rounded-lg font-semibold transition-all shadow-lg shadow-yellow-500/30 text-sm"
>
  <Crown className="w-5 h-5" />
  Premium
</button>
```

#### 4. Login Butonu Görünüm İyileştirmesi ✅
**ÖNCE:** Çok sade, gri hover
```jsx
className="text-gray-700 hover:bg-gray-100"
```

**SONRA:** Outline stil, şık ve görünür
```jsx
className="border-2 border-primary-600 text-primary-600 hover:bg-primary-50 font-semibold"
```

**Değişiklikler:**
- ✅ `border-2 border-primary-600` eklendi (mor çerçeve)
- ✅ `hover:bg-primary-50` eklendi (mor hover arka plan)
- ✅ `font-semibold` eklendi (kalın yazı)
- ✅ `text-left` kaldırıldı (ortalandı)

#### 5. Register Butonu Görünüm İyileştirmesi ✅
**ÖNCE:** Gradient dolu (ana CTA gibi)
```jsx
className="bg-gradient-to-r from-primary-500 to-primary-600"
```

**SONRA:** Outline stil (Login ile uyumlu)
```jsx
className="border-2 border-blue-600 text-blue-600 hover:bg-blue-50"
```

**Sebep:** Login ile benzer stil, ama farklı renk (mavi vs mor)
