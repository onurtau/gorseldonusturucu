### 🐛 2. WATERMARKPREVIEW HATA DÜZELTMELERİ

#### A) Önizleme Bug Fix - Store Yapısı Değişikliği

**Sorun:**  
Store yapısı değişti: `files` → `files[ui.activeView]`  
WatermarkPreview hâlâ eski yapıyı kullanıyordu → Önizleme gösterilmiyordu

**Çözüm:**
```javascript
// src/components/WatermarkPreview.js

// ❌ ESKİ (hatalı):
const { files, conversionSettings } = useAppStore();
if (!files || files.length === 0) return null;

// ✅ YENİ (düzeltilmiş):
const { files, conversionSettings, ui } = useAppStore();
const activeFiles = files[ui.activeView] || [];
if (!activeFiles || activeFiles.length === 0) return null;

// useEffect dependency güncellendi:
useEffect(() => {
  const firstFile = activeFiles[0];
  // ... preview URL oluştur
}, [activeFiles[0]?.id, activeFiles[0]?.file?.name, activeFiles[0]?.path]);
```

**Değişiklikler:**
- Satır 5: `ui` eklendi store'dan
- Satır 12: `activeFiles = files[ui.activeView]` 
- Satır 54-69: useEffect içinde `activeFiles` kullanımı
- Satır 104: Conditional render `activeFiles` kontrolü

#### B) React Hook Kuralları Hatası

**Hata:**
```
ERROR: Rendered more hooks than during the previous render.
```

**Sorun:**  
`useMemo` hook'u koşullu return'den **sonra** çağrılıyordu (React kuralı ihlali)

**Çözüm:**
```javascript
// ❌ ESKİ (hatalı sıra):
useEffect(() => { ... });
if (!activeFiles) return null; // Koşullu çıkış
const tiledWatermarks = useMemo(() => { ... }); // ❌ Hook koşuldan sonra!

// ✅ YENİ (doğru sıra):
useEffect(() => { ... });
const tiledWatermarks = useMemo(() => { ... }); // ✅ Hook koşuldan ÖNCE
if (!activeFiles) return null; // Koşullu çıkış en sonda
```

**React Kuralı:**  
> Hook'lar her render'da aynı sırada çalışmalı. Koşullara, döngülere veya return'lerden sonra konulmamalı.

#### C) Performans Optimizasyonu

**Problem:**  
- Her render'da yüzlerce tiled watermark yeniden hesaplanıyordu
- Dosya değişmediğinde bile preview URL tekrar oluşturuluyordu

**Çözüm 1: Akıllı Dependency**
```javascript
// ❌ ESKİ:
useEffect(() => {
  // Preview URL oluştur
}, [activeFiles]); // Tüm activeFiles değiştiğinde

// ✅ YENİ:
useEffect(() => {
  // Preview URL oluştur
}, [activeFiles[0]?.id, activeFiles[0]?.file?.name, activeFiles[0]?.path]);
// Sadece ilk dosya değiştiğinde
```

**Çözüm 2: Tiled Watermarks Memoization**

  }

}, [
  conversionSettings.watermark.text,
  conversionSettings.watermark.tilePattern,

#### B) ImageMagick Dönüşüm Fonksiyonu
```javascript
async function convertWithImageMagick(inputPath, outputPath, format, quality) {
  try {
    let command;
    
    if (format === 'jpeg' || format === 'jpg') {
      command = `magick convert "${inputPath}" -quality ${quality} "${outputPath}"`;
    } else if (format === 'tiff' || format === 'tif') {
      command = `magick convert "${inputPath}" -compress lzw -quality ${quality} "${outputPath}"`;
    }
    
    console.log('ImageMagick komutu:', command);
    execSync(command, { stdio: 'pipe' });
    
    const originalStats = await fs.stat(inputPath);
    const newStats = await fs.stat(outputPath);
    
    console.log('✓ ImageMagick ile CMYK dönüşümü başarılı');
    
    return {
      success: true,
      outputPath,
      originalSize: originalStats.size,
      newSize: newStats.size,
      compressionRatio: ((1 - newStats.size / originalStats.size) * 100).toFixed(2),
      method: 'ImageMagick (CMYK korundu)'
    };
  } catch (error) {
    console.error('ImageMagick hatası:', error.message);
    throw error;
  }
}
```

#### 🔄 Senkronizasyon Stratejisi

**Cache-First:**
- İlk yükleme: Supabase'den çek
- Sonraki yüklemeler: Cache'den göster
- Background'da: Supabase'den güncelle
- Conflict: Son yazma kazanır

**Queue System:**
- FIFO (First In First Out)
- Retry: Max 3 kez
- Exponential backoff (1s, 2s, 4s)
- Failed items loglanır

const t = (key, params = {}) => {

async function convertWithImageMagick(inputPath, outputPath, format, quality, targetColorSpace = null) {
  try {
    const formatUpper = format.toUpperCase();
    let command;
    let colorSpaceArg = '';
    
    // ✨ Renk uzayı dönüşümü parametresi
    if (targetColorSpace === 'cmyk') {
      colorSpaceArg = '-colorspace CMYK';
      console.log('ImageMagick: RGB → CMYK dönüşümü yapılacak');
    } else if (targetColorSpace === 'rgb' || targetColorSpace === 'srgb') {
      colorSpaceArg = '-colorspace sRGB';
      console.log('ImageMagick: CMYK → RGB dönüşümü yapılacak');
    }
    
    if (formatUpper === 'JPEG' || formatUpper === 'JPG') {
      command = `magick convert "${inputPath}" ${colorSpaceArg} -quality ${quality} "${outputPath}"`;
    } else if (formatUpper === 'TIFF' || formatUpper === 'TIF') {
      command = `magick convert "${inputPath}" ${colorSpaceArg} -compress lzw -quality ${quality} "${outputPath}"`;
    }
    
    execSync(command, { stdio: 'pipe' });
    
    const conversionMethod = targetColorSpace 
      ? `ImageMagick (${targetColorSpace.toUpperCase()} dönüşümü)` 
      : 'ImageMagick (renk uzayı korundu)';
    console.log('✓', conversionMethod, 'başarılı');
    
    return {

    };
  } catch (error) {

    throw error;
  }
}
