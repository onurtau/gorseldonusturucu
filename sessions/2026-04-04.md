### ⏰ Session 4: Web Versiyonu Olası Sorunlar (00:25-00:30)

**🎯 Kullanıcı Sorusu:**
> "Web sürümünde oluşacak olası sorunlar nedir? Veya bir sorun oluşmaz mı?"

#### ⚠️ Olası Sorunlar ve Çözümleri

**1. CORS (Cross-Origin Resource Sharing)**

**Sorun:**
```javascript
// Frontend: localhost:3000
// Backend: localhost:5000
// Tarayıcı: "CORS policy blocked"
```

**Çözüm:**
```javascript
// backend/server.js
const cors = require('cors');

app.use(cors({
  origin: ['http://localhost:3000', 'https://yourdomain.com'],
  credentials: true
}));
```

**Durum:** ✅ Çözülebilir (15 dakika)

---

**2. Dosya Boyutu Limitleri**

**Sorun:**
```javascript
// Default Express limit: 1MB
// Kullanıcı 50MB dosya yüklerse: Request Entity Too Large
```

**Çözüm:**
```javascript
// backend/server.js
app.use(express.json({ limit: '50mb' }));
app.use(express.urlencoded({ limit: '50mb', extended: true }));

// Multer
const upload = multer({ 
  limits: { fileSize: 50 * 1024 * 1024 } // 50MB
});
```

**Durum:** ✅ Çözülebilir (10 dakika)

---

**3. Timeout (İşlem Süresi Aşımı)**

**Sorun:**
```javascript
// 100 dosya toplu işlem
// Server timeout: 30 saniye
// İşlem 2 dakika sürer → Timeout error
```

**Çözüm:**
```javascript
// backend/server.js
const server = app.listen(5000);
server.timeout = 300000; // 5 dakika

// Alternatif: Queue system (Bull)
const queue = new Bull('image-processing');
queue.process(async (job) => {
  // Background işleme
});
```

**Durum:** ✅ Çözülebilir (Queue system: 1-2 saat)

---

**4. Server RAM (Bellek Doluluğu)**

**Sorun:**
```javascript
// 10 kullanıcı aynı anda 20MB dosya yüklerse
// RAM: 512MB (free tier)
// 10 * 20MB = 200MB + Sharp işleme = 400-500MB
// Server crash!
```

**Çözüm:**
```javascript
// Rate limiting
const rateLimit = require('express-rate-limit');

const limiter = rateLimit({
  windowMs: 15 * 60 * 1000, // 15 dakika
  max: 100 // 100 istek/15dk
});

app.use('/api/', limiter);

// Queue + Worker Pool
// Aynı anda max 3-5 işlem
```

**Durum:** ✅ Çözülebilir (Rate limit: 30 dakika)

---

**5. Cold Start (İlk İstek Yavaşlığı)**

**Sorun:**
```javascript
// Serverless (Vercel, Railway free tier)
// 10 dakika kullanılmazsa server sleep
// İlk istek: 5-10 saniye wakeup süresi
```

**Çözüm:**
```javascript
// Ping endpoint (keep-alive)
setInterval(() => {
  fetch('https://api.yourdomain.com/ping');
}, 5 * 60 * 1000); // 5 dakikada bir

// Veya: Always-on dedicated server ($12/ay)
```

**Durum:** ⚠️ Serverless'da kaçınılmaz, dedicated server ile çözülür

---

**6. File Cleanup (Geçici Dosya Birikimi)**

**Sorun:**
```javascript
// Her işlemde uploads/ klasörüne dosya yazılır
// 1000 işlem = 1000 dosya
// Disk dolar, server crash
```

**Çözüm:**
```javascript
// İşlem sonrası temizleme
const fs = require('fs').promises;

app.post('/api/convert', upload.single('file'), async (req, res) => {
  try {
    const result = await sharp(req.file.path).toFormat('png').toBuffer();
    res.send(result);
  } finally {
    // Dosyayı temizle
    await fs.unlink(req.file.path).catch(() => {});
  }
});

// Ek: Cron job (günlük temizlik)
const cron = require('node-cron');
cron.schedule('0 0 * * *', async () => {
  // uploads/ klasörünü temizle
});
```

**Durum:** ✅ Çözülebilir (30 dakika)

---

**7. Rate Limiting (Spam Koruması)**

**Sorun:**
```javascript
// Kötü niyetli kullanıcı
// 1000 istek/dakika atar
// Server çöker veya maliyet patlar
```

**Çözüm:**
```javascript
// IP-based rate limiting
const rateLimit = require('express-rate-limit');

const apiLimiter = rateLimit({
  windowMs: 1 * 60 * 1000, // 1 dakika
  max: 10, // 10 istek/dakika
  message: 'Çok fazla istek. Lütfen bekleyin.',
  standardHeaders: true,
  legacyHeaders: false,
});

app.use('/api/', apiLimiter);

// Premium kullanıcılar için daha yüksek limit
const premiumLimiter = rateLimit({
  windowMs: 1 * 60 * 1000,
  max: 100, // 100 istek/dakika
  skip: (req) => req.user?.isPremium
});
```

**Durum:** ✅ Çözülebilir (45 dakika)

**Neden Gerekli:**


**Kurulum:**

```bash

```

**Kod:**

```javascript
// queues/imageQueue.js
const Queue = require('bull');
const sharp = require('sharp');

const imageQueue = new Queue('image-processing', {
  redis: {
    host: '127.0.0.1',
    port: 6379
  },
  limiter: {
    max: 5, // Aynı anda max 5 job
    duration: 1000 // 1 saniyede
  }
});

// Worker (işi yapan)
imageQueue.process(4, async (job) => {
  const { filePath, format, quality } = job.data;
  
  // Progress update
  job.progress(10);
  
  // İşleme
  const result = await sharp(filePath)
    .toFormat(format, { quality })
    .toBuffer();
  
  job.progress(100);
  
  return { success: true, buffer: result };
});

// Event listeners
imageQueue.on('completed', (job, result) => {
  console.log(`Job ${job.id} completed`);
  // WebSocket ile kullanıcıya bildir
});

imageQueue.on('failed', (job, err) => {
  console.error(`Job ${job.id} failed:`, err);
  // Retry mechanism
});

module.exports = imageQueue;
```

**API Endpoint:**

```javascript
// routes/api.js
const imageQueue = require('../queues/imageQueue');

app.post('/api/convert-bulk', async (req, res) => {
  const { files } = req.body;
  
  // Job'ları queue'ya ekle
  const jobs = await Promise.all(
    files.map(file => imageQueue.add({
      filePath: file.path,
      format: 'png',
      quality: 90
    }, {
      attempts: 3, // 3 kez retry
      backoff: 5000 // 5 saniye bekle
    }))
  );
  
  // Job ID'lerini döndür
  res.json({
    success: true,
    jobIds: jobs.map(j => j.id)
  });
});

// Progress endpoint
app.get('/api/job/:id/progress', async (req, res) => {
  const job = await imageQueue.getJob(req.params.id);
  const progress = await job.progress();
  const state = await job.getState();
  
  res.json({ progress, state });
});
```

**Avantajları:**
- ✅ Toplu işlem çökme riski YOK
- ✅ Progress tracking (kullanıcı ne kadar tamamlandı görebilir)
- ✅ Retry mechanism (hata durumunda tekrar dene)

**Backend (Queue):**
```javascript
// routes/convert.js
app.post('/api/convert-bulk', async (req, res) => {
  const { files, userId } = req.body;
  
  // Batch job oluştur
  const batchId = uuidv4();
  
  // Her dosya için queue job
  const jobs = await Promise.all(
    files.map((file, index) => 
      imageQueue.add({
        batchId,
        userId,
        fileIndex: index,
        totalFiles: files.length,
        filePath: file.path,
        format: file.format,
        quality: file.quality
      }, {
        attempts: 3,
        priority: req.user.isPremium ? 1 : 10 // Premium öncelikli
      })
    )
  );
  
  res.json({
    success: true,
    batchId,
    jobCount: jobs.length
  });
});
```

**Worker (Processing):**
```javascript
// queues/imageQueue.js
imageQueue.process(4, async (job) => {
  const { batchId, fileIndex, totalFiles, filePath, format, quality } = job.data;
  
  // Progress: 0-100
  job.progress(0);
  
  // İşleme
  const result = await sharp(filePath)
    .toFormat(format, { quality })
    .toBuffer();
  
  // File cleanup
  await fs.unlink(filePath);
  
  job.progress(100);
  
  // WebSocket'e publish
  io.to(batchId).emit('file-completed', {
    fileIndex,
    totalFiles,
    progress: Math.round((fileIndex + 1) / totalFiles * 100)
  });
  
  return { success: true, buffer: result };
});
```

**Frontend (WebSocket):**
```javascript
// React component
import io from 'socket.io-client';

const [progress, setProgress] = useState(0);
const [completedFiles, setCompletedFiles] = useState([]);

useEffect(() => {
  const socket = io('https://api.yourdomain.com');
  
  socket.emit('join-batch', batchId);
  
  socket.on('file-completed', (data) => {
    setProgress(data.progress);
    setCompletedFiles(prev => [...prev, data.fileIndex]);
    
    if (data.progress === 100) {
      // Tüm dosyalar tamamlandı
      toast.success('Tüm dosyalar işlendi!');
    }
  });
  
  return () => socket.disconnect();
}, [batchId]);

return (
  <div>
    <ProgressBar value={progress} />
    <p>{completedFiles.length} / {totalFiles} dosya tamamlandı</p>
  </div>
);
```
