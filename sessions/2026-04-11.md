## 6. Özellikler ve Teknik Detaylar

### 6.1. Akıllı Navigation Sistemi

**LicenseModal Navigation Mantığı:**
```javascript
const handleTermsClick = (e) => {
  e.preventDefault();
  if (isAuthenticated) {
    toggleLicenseModal();
    setActiveView('terms');
  } else {
    window.location.hash = 'terms';
    toggleLicenseModal();
  }
};
```

**Avantajlar:**
- ✅ Platform bağımsız (Web + Electron)
- ✅ Kullanıcı durumuna göre otomatik adaptasyon
